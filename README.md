# Recipe-Creator

A Recipe-Creator Website build with NextJS, AppRouter Mode, uses SSR for most Parts. [`Neon`](https://neon.com/) is used for DB and Auth, [`Pinata`](https://pinata.cloud) for Image Storage.

## Getting Started

First, clone the repo:

```bash
git clone git@github.com:stackSm4shr/Recipe-Creator.git
cd Recipe-Creator
```

Install dependencies:

```bash
npm install
```

create .env file:

you have to signup to [`Vercel`](https://vercel.com/)/[`Neon`](https://neon.com/)/[`Pinata`](https://pinata.cloud) and get your credentials,
take sample.env.local file as a start, input required data and save as  .env.local inside projects rootfolder

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
